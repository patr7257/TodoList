Build / Test: 
mvn -pl server test
mvn -pl client test

RUN / START 
Run server: mvn -pl server exec:java        (from repo root)
Run client: mvn -pl client javafx:run       (from repo root)








mvn clean test (from client or server folders)

"cd server; mvn clean test"

Prerequisites
-------------
- Install JDK 17
- Install Apache Maven ("mvn" on PATH)

Check in PowerShell
-------------------
- java -version
- mvn -version

Build / Test (from repo root)
----------------------------
- mvn clean test
- If you just want a build artifact: mvn clean package


RUN PROGRAMS
------------
Open two terminals (recommended): one for server, one for client.

Server (from repo root):
- mvn -pl server exec:java

Client (from repo root):
- mvn -pl client javafx:run

Alternative (run from module folders):
- cd server; mvn exec:java
- cd client; mvn javafx:run
